# ethden-admin-js
## Install
`npm install`

## Start
`node app.js`

This will start the server locally.

`GET`: `/vendors/`: Returns all the vendors with their payouts
`PUT`: `/vendors/id`: 
`GET`: `/transactions`: Returns all the transactions
`PUT`: `/addTx:`: Adds Tx into the db
